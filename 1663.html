Pattern 52

I’m calling to…
Meaning : ...ခေါ်လိုက်တာ(ဖုန်းပြောခြင်း)။

Example :
I’m calling to see if you have the latest PES game in stock.
ခင်ဗျားဆီမှာ PES နောက်ဆုံးထွက်ဂိမ်းလေးရှိမလားလိုခေါ်လိုက်တာ။

I’m calling to ask you to fix my computer.
ကွန်ပျူတာ ဘယ်လိုပြင်ရမလဲလိုမေးမလို မင်းကိုခေါ်လိုက်တာ။

I’m calling to complain about the noise from my neighbors house.
ဘေးအိမ်က ဆူညံသံတွေတိုင်မလို ခေါ်လိုက်တာပါ။

I’m calling to order some food.
အစားအသောက်တစ်ချို မှာချင်လိုခေါ်လိုက်တာပါ။

I'm calling to talk with John.
ဂျွန်နဲ့ ပြောချင်လိုခေါ်လိုက်တာပါ။
