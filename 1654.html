Pattern 43

I’d be grateful if you could...
Meaning : ...ကောင်းတာပေါ့(ယဉ်ကျေးစွာပြောခြင်း)။

Example :
I would be grateful if you could tell me anything about the guitar.
မင်းငါ့ကို ဂစ်တာအကြောင်းပြောပြရင် သိပ်ကောင်းမှာပဲ။
We would be grateful if you could return the signed contracts.
မင်းရဲ့ စာချုပ်ကို ငါတိုဆီပြန်ပေးနိုင်ရင် ကောင်းမှာပဲ။
I'd be grateful if you could return my money.
ငါ့ပိုက်ဆံ ပြန်ပေးရင် သိပ်ကောင်းမှာပဲ။
