Pattern 14

Shouldn't we…?
...သင့်တယ်မဟုတ်လား။

Example :

Shouldn’t we ask for permission first? ငါတို ခွင့်ပြုချက် အရင်တောင်းသင့်တယ် မဟုတ်လား။

Shouldn’t we read the instructions before we install? ငါတို ဒါမတပ်ဆင်ခင် ညွှန်ကြားချက်တွေ အရင်ဖတ်သင့်တယ် မဟုတ်လား။

Shouldn’t we check the weather before we go out? အပြင်မထွက်ခင် ငါတို ရာသီဥတု အခြေအနေ ကြည့်သင့်တယ် မဟုတ်လား။

Shouldn’t you try to save some money rather than spend it all? မင်းပိုက်ဆံတွေအကုန်သုံးလိုက်မဲ့အစား စုထားသင့်တယ် မဟုတ်လား။

Shouldn't we study first? အရင်ဆုံး စာလုပ်သင့်တယ် မဟုတ်လား။
