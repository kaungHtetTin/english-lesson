Pattern-78

There’s still some… left
...အနည်းငယ်ကျန်သေးတယ်။

Example:

There’s still some cheese left in the refrigerator.
ရေခဲသေတ္တာထဲ ချိစ့်အနည်းငယ် ကျန်သေးတယ်။

There’s still some time left to catch the plane if we hurry.
ငါတိုသာမြန်မြန်လုပ်ရင် လေယာဉ်မှီဖို အချိန်နည်းနည်း ကျန်သေးတယ်။

There’s still some good left in the world.
ကမ္ဘာကြီးမှာ ကောင်းတာတစ်ချို ကျန်ပါသေးတယ်။

There are still some people left in the classroom.
စာသင်ခန်းထဲ လူတစ်ချို ကျန်နေသေးတယ်။

There's still some jobs left you can find.
မင်းရှာလိုရတဲ့ အလုပ်တစ်ချိုကျန်ပါသေးတယ်။
