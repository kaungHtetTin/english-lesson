Pattern 48

If it hadn’t been for…
Meaning : ...သာမဟုတ်ရင်။

Example :
If it hadn’t been for your help, my experiments would have failed.
မင်းအကူအညီကြောင့်သာမဟုတ်ရင် ငါ့စမ်းသပ်မှုက ကျဆုံးလောက်တယ်။

If it hadn’t been for the lifeboat, I would have drowned.
အသက်ကယ်လှေကြောင့်သာမဟုတ်ရင် ငါရေနစ်နေလောက်ပြီ။

If it hadn’t been for her arrangement, the party would not have been a success.
သူမရဲ့ အစီအစဉ်တွေကြောင့်သာမဟုတ်ရင် ပါတီက အဲ့လောက်အောင်မြင်မှာ မဟုတ်ဘူး။

If it hadn't been for my parents I wouldn't been have born.
ငါ့မိဘတွေကြောင့်သာမဟုတ်ရင် ငါမွေးခဲ့မှာမဟုတ်ဘူး။

She'd never noticed if it hadn't been for Sid.
ဆစ်ဒ်ကြောင့်သာမဟုတ်ရင် သူမသတိထားမိမှာတောင်မဟုတ်ဘူး။
