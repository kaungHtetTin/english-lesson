စာသင်ခန်းရှိပစ္စည်းများ
An + Vowel sound (သရသံများ)
A + Consonant sound (ဗျည်းသံများ)

ကျေးဇူးပြု၍ အသံဖိုင်ကို အရင်နားဆင်ပါ

1. This is a book.
ဤအရာသည် စာအုပ်တစ်အုပ်ဖြစ်ပါသည်။
2. This is an English book.
ဤအရာသည် အင်္ဂလိပ်စာ စာအုပ်တစ်အုပ်ဖြစ်ပါသည်။
3. This is an eraser.
ဤအရာသည် ခဲဖျက်တစ်တုံးဖြစ်ပါသည်။
4. This is a notebook.
ဤအရာသည် မှတ်စုစာအုပ်တစ်အုပ်ဖြစ်ပါသည်။
5. This is a pen.
ဤအရာသည် ဘောပင်တစ်ချောင်းဖြစ်ပါသည်။
6. This is a clock.
ဤအရာသည် နာရီတစ်လုံးဖြစ်ပါသည်။
